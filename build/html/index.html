Action client node

This node implements an action client that allows the user to set a target (x, y) or to cancel it. It interact with the user allowing to cancel the goal, receive feedback or close the node. It also publishes the robot position and velocity as a custom message (x,y, vel_x, vel_z), by relying on the values published on the topic /odom.

Module author: Chiara Buono <s7687956@studenti.unige.it>
